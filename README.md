## 📖 Overview

PokeSearch is a modern, single-page application (SPA) that allows users to instantly search and retrieve detailed statistics for thousands of Pokémon. Designed with a focus on user experience, the application features an intuitive auto-suggest dropdown that populates in real-time as users type, eliminating the frustration of exact-spelling requirements. 

## ✨ Key Features

- **Live Search Suggestions:** Implements a dynamic auto-complete dropdown. The app efficiently caches Pokémon names on the initial load to provide instantaneous, typeahead search suggestions.
- **Comprehensive Pokémon Stats:** Fetches and beautifully presents data from the Pokémon database, including:
  - Base Experience
  - Height & Weight
  - Elemental Types
  - High-quality Sprite Imagery
- **Robust Error Handling:** Gracefully handles invalid searches and network errors, providing clear user feedback when a Pokémon cannot be found.
- **Fully Responsive UI:** Built with a mobile-first approach using Tailwind CSS, ensuring a seamless experience across desktops, tablets, and mobile devices.
- **Modern React Architecture:** Utilizes functional components, React Hooks (`useState`, `useEffect`, `useMemo`), and efficient state management.

## 🛠️ Technology Stack

- **Frontend Framework:** React (v19)
- **Build Tool:** Vite (v7)
- **Styling:** Tailwind CSS (v4) & Custom CSS
- **Icons:** FontAwesome
- **Data Source:** [PokeAPI (RESTful API)](https://pokeapi.co/)
