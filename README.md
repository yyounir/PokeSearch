# PokeSearch - PokeAPI Search Engine
Search and find stats on your favorite Pokemon!

## Features
<ul>
  <li>Uses PokeAPI to find and search for thousands of Pokemon</li>
  <li>Don't know how to spell it? You can see search suggestions as you type if you're looking for that specific pokemon!</li>
  <li>See the types of Pokemon as you search up and view their experience and the weight of each Pokemon from their Pokemon database</li>
  <li>Mobile friendly</li>
</ul>

## Coding Languages / Frameworks Used
<ul>
  <li>React/Vite</li>
  <li>CSS</li>
  <li>JavaScript</li>
  <li>Tailwind CSS</li>
  <li>RESTful API</li>
</ul>
